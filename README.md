# GetEventListener
webservice to fetch events attached to any non native html element

## Requirement
* nodejs v10.16.3
* npm 6.9.0

## Usage

### Install node module
``` npm install ```

### Start Server
```npm start```

### Execute tests
``` npm test ```

## API docs 
webservice docs can be found once the server is started at http://localhost:3000/api-docs/#/
